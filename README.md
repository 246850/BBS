# BBS
基于asp.net core mvc 3.1/net core3.1 EfCore3编写的轻量级社区论坛系统
### 配置
```
{
  "Logging": {
    "LogLevel": {
      "Default": "Information",
      "Microsoft": "Warning",
      "Microsoft.Hosting.Lifetime": "Information"
    }
  },
  "AllowedHosts": "*",
  "ConnectionStrings": {
    "MySQL": "server=localhost;user id=root;password=123456; database=bbsdb; charset=utf8"
  },
  "Sites": {
    "Title": "XXX论坛",
    "From": 2019
  }
}
```
### 数据库MySql执行文件(bbsdb.sql)
